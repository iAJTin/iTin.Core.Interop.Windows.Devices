# DEVMODE.dmFields field

Especifica si algunos miembros de la estructura DEVMODE se ha inicializado. Si un miembro se ha inicializado, su correspondiente bit está establecido a '1', de lo contrario el bit es '0'. Un controlador admite sólo a los miembros DEVMODE que son adecuados para la impresora o dispositivo de visualización. Para más información ver [`DM`](../DEVMODE.DM.md).
