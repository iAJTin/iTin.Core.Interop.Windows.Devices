# DEVMODE.dmCollate field

Especifica si se debe utilizar intercalación al imprimir varias copias. (Este campo se omite a menos que el controlador de impresora establezca el campo DM_COLLATE). Este miembro puede ser uno de los siguientes valores. DMCOLLATE_TRUE, intercalar al imprimir varias copias. DMCOLLATE_FALSE. no intercalar al imprimir varias copias.
