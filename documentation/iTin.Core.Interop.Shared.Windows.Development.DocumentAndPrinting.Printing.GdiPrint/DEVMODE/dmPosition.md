# DEVMODE.dmPosition field

Para dispositivos de pantalla sólo, una estructura [`POINTL`](../../iTin.Core.Interop.Shared.Windows.Development.Graphics.Legacy.Gdi.Metafiles/POINTL.md) que indica las coordenadas posicionales de la pantalla en referencia a la zona de escritorio. El dispositivo de pantalla principal siempre está situado en las coordenadas (0,0).
