# DEVMODE.dmDisplayFlags field

Especifica el modo de visualización del dispositivo. Este miembro puede ser una combinación de los siguientes valores. DM_GRAYSCALE, especifica que la pantalla es un dispositivo monocromo. Si no se establece este indicador, se asume que es de color. DM_INTERLACED, especifica que el modo de visualización es entrelazado. Si no se establece este indicador, se asume que es no entrelazado.
