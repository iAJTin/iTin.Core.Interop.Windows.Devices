# NativeMethods.EnumDisplaySettingsEx method

Retrieves information about one of the graphic modes for a display device. To retrieve information for all graphic modes for a display device, make a series of calls to this function. This function differs from [`EnumDisplaySettings`](./EnumDisplaySettings.md) in that there is a dwFlags parameter. For more information, please see http://msdn.microsoft.com/en-us/library/dd162612.aspx.
