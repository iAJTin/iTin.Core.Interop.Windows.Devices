public enum ChangeDisplaySettingsFlags

## Values

| name | value | description |
| --- | --- | --- |
| CDS_UPDATEREGISTRY | `0x1` | Change the current graphic mode, this will be updated in the registry. The mode information is stored in the user profile. |
| CDS_TEST | `0x2` | The system checks if the requested graphic mode can be set. |
| CDS_FULLSCREEN | `0x4` | The mode is temporary. If you switch to and from another desktop, this mode will not be reset. |
| CDS_GLOBAL | `0x8` | The configuration will be saved in the global configuration area so that it will affect all users of the machine. Otherwise, only the user settings are modified. This indicator is only valid when used together with the flag CDS_UPDATEREGISTRY. |
| CDS_SET_PRIMARY | `0x10` | This device will become the primary device. |
| CDS_VIDEOPARAMETERS | `0x20` | When set, the  lParam  parameter is a pointer to a VIDEOPARAMETERS structure. |
| CDS_NORESET | `0x10000000` | The settings are saved in the registry, but will not take effect. This indicator is only valid when used together with the flag CDS_UPDATEREGISTRY. |
| CDS_RESET | `0x40000000` | You must change the settings, even if the requested settings are the same as the current settings. |
