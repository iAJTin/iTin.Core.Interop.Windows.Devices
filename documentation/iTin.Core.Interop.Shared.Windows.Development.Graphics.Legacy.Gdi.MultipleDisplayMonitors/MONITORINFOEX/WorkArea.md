A structure [`RECT`](../../iTin.Core.Interop.Shared.Windows.Development.Graphics.Legacy.Gdi.Rectangles/RECT.md) that specifies the rectangle of the working area of the useful monitor that can be used by applications, expressed in virtual screen coordinates.
